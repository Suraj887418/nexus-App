# 🏢 NEXUS CORP - Smart Employee Management System

![Nexus Corp Banner](https://images.unsplash.com/photo-1497215728101-856f4ea42174?auto=format&fit=crop&q=80&w=1200&h=400)

Welcome to the **NEXUS CORP Smart Employee Management System**, a complete Full-Stack solution for modern enterprise management. This project includes a highly interactive React Native mobile application for employees and a powerful React-based Admin Dashboard for HR and management.

---

## ✨ Key Features

### 📱 Employee Mobile App (React Native / Expo)
*   🤖 **AI Face Recognition Attendance:** Employees check in/out using live face verification.
*   📍 **Location Tracking:** Ensures employees are within the allowed geofence during check-ins.
*   🚨 **SOS Emergency Button:** Instantly alerts HR and management with the employee's live location.
*   📊 **Real-time Task Management:** View assigned cases, update status, and manage daily workflows.
*   📅 **Leave Management:** Request leaves and view current leave balances.

### 💻 Admin Dashboard (React / Vite)
*   📈 **Real-time Analytics:** Visual representation of employee attendance, case progress, and alerts.
*   👥 **Employee Management:** Add, edit, and track employees effortlessly.
*   🔔 **Live Notifications:** Receive instant SOS alerts and leave requests.
*   ⚙️ **Global Settings:** Configure check-in times and company policies.

### ⚙️ Backend (Node.js / Express)
*   🧠 **AI Processing:** Server-side face matching using `face-api.js`.
*   🗄️ **MongoDB Database:** Secure and scalable data storage.
*   🔒 **JWT Authentication:** Secure API access for both mobile and web clients.
